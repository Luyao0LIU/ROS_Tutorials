# 深度学习：欠拟合问题的几种解决方案
现象：在训练的过程中损失值loss依然很大，或者几乎不变、严格不变，在这种情况下被称为是"欠拟合问题"。
神经网络欠拟合的特征是，训练了很长时间，但是在训练集上，loss值仍然很大甚至与初始值没有太大区别，而且精确度也很低，几乎接近于0，在测试集上亦如此。且先不管模型结构配置的优劣，就欠拟合问题来说，需要从如下方面来着手。
